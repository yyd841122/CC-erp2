# 🎨 设计规范文档 (DESIGN_SPEC.md)

## 1. 设计理念

- **关键词**: 简约、科技、高效、专业
- **设计风格**: 结合简约现代与科技感的企业级设计
- **核心原则**:
  - 信息密度适中，避免视觉过载
  - 操作流程清晰，3步内完成单据录入
  - 无多余动画，注重响应速度
  - 数据展示直观，支持快速决策

## 4. 交互细节

### 4.1 按钮交互

| 类型 | 默认 | Hover | Active | Disabled |
|:---|:---|:---|:---|:---|
| **主按钮** | 背景 `#1890FF`，文字白色 | 背景 `#40A9FF` | 背景 `#096DD9` | 背景 `#F5F5F5`，文字 `#BFBFBF` |
| **次按钮** | 边框 1px solid `#D9D9D9` | 边框 `#1890FF`，文字 `#1890FF` | 背景 `#E6F7FF` | 背景 `#F5F5F5` |
| **文字按钮** | 文字 `#1890FF` | 文字 `#40A9FF` | 文字 `#096DD9` | 文字 `#BFBFBF` |

**交互反馈**:
- 点击时无缩放动画（按要求）
- 状态切换即时响应，无过渡延迟
- 加载状态显示 Loading 图标在按钮内部

### 4.2 表单验证

| 状态 | 样式 |
|:---|:---|
| **默认** | 边框 `#D9D9D9` |
| **校验中** | 边框 `#1890FF`，显示 Loading 图标 |
| **成功** | 边框 `#52C41A`，右侧显示 ✓ 图标 |
| **错误** | 边框 `#F5222D`，下方显示错误提示文字 |

**错误提示**:
- 位置：输入框下方，字号 12px，颜色 `#F5222D`
- 无抖动动画（按要求）

### 4.3 加载状态

| 场景 | 方案 |
|:---|:---|
| **按钮加载** | 按钮内显示旋转 Loading 图标，文字变为"加载中..." |
| **页面加载** | 使用骨架屏，灰色条纹动画 |
| **表格加载** | 表格区域显示骨架屏，模拟表格行 |
| **图表加载** | 图表区域显示 Loading 图标 + "加载中..." 文字 |

**骨架屏样式**:
- 背景 `linear-gradient(90deg, #F2F2F2 25%, #E6E6E6 50%, #F2F2F2 75%)`
- 动画时长 1.5s，无限循环

### 4.4 弹窗与抽屉

| 类型 | 用途 | 尺寸 |
|:---|:---|:---|
| **Dialog** | 表单确认、简单信息展示 | 宽度 520px / 720px |
| **Drawer** | 复杂表单、详情页编辑 | 宽度 640px / 800px，从右侧滑出 |

**无动画过渡**（按要求）: 直接显示/隐藏，无滑入滑出效果
